# LLM Response Consistency & Reliability Checker

## Overview
This project investigates hallucination behavior in language models by analyzing response consistency across multiple prompt variations.

## Core Idea
Instead of relying on a single response, the system:
- Generates multiple prompts
- Collects multiple outputs
- Detects contradictions

## Why This Matters
Hallucinations often appear as inconsistent answers rather than obviously false ones. This system identifies such patterns.

## Features
- Multi-prompt generation
- Consistency-based hallucination detection
- Lightweight evaluation framework
